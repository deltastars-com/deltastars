# دليل تحويل متجر الأقمشة إلى تطبيق (Android & iOS) 📱

لقد تم إعداد المتجر بأعلى معايير الجودة ليكون جاهزاً للتحويل الفوري إلى تطبيق أصلي (Native App) باستخدام تقنية **Capacitor**.

## 1. المتطلبات الأساسية
تأكد من تثبيت الأدوات التالية على جهازك:
- **Node.js** (إصدار 18 أو أحدث)
- **Android Studio** (لإصدار أندرويد)
- **Xcode** (لإصدار آيفون - يتطلب جهاز ماك)

## 2. خطوات التحويل (أندرويد APK)

### الخطوة الأولى: تثبيت Capacitor
افتح الطرفية (Terminal) في مجلد المشروع وقم بتشغيل:
```bash
npm install @capacitor/core @capacitor/cli
npx cap init "Delta Stars" "com.deltastars.app" --web-dir dist
```

### الخطوة الثانية: إضافة منصة أندرويد
```bash
npm run build
npx cap add android
```

### الخطوة الثالثة: مزامنة الملفات
في كل مرة تقوم فيها بتعديل الكود، قم بتشغيل:
```bash
npm run build
npx cap copy
```

### الخطوة الرابعة: إنشاء الـ APK
1. افتح مجلد `android` باستخدام **Android Studio**.
2. انتظر تحميل الـ Gradle.
3. اذهب إلى القائمة: `Build` > `Build Bundle(s) / APK(s)` > `Build APK(s)`.
4. مبروك! ستحصل على ملف الـ APK جاهزاً للتثبيت.

## 3. مميزات التطبيق الحالية
- **العمل بدون إنترنت (Offline Mode):** بفضل الـ Service Worker المطور.
- **الإشعارات اللحظية:** دعم كامل لإشعارات الـ Push.
- **تتبع المندوب:** نظام تتبع حي باستخدام الـ GPS.
- **تسجيل دخول حيوي:** دعم البصمة والوجه (Biometrics).
- **أداء فائق:** المتجر مصمم ليعمل بسرعة 60 إطار في الثانية.

## 4. سياسة الخصوصية والاستخدام
تم تضمين صفحات السياسات داخل التطبيق لتتوافق مع متطلبات **Google Play** و **Apple App Store**. يمكنك الوصول إليها من تذييل الصفحة (Footer).
